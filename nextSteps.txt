### Missing Tests
A vast majority of tests are missing. The Python test suite for `Interval` is minimal compared to the MATLAB version.
- **Existing Python tests cover:** `interval` (constructor), `and_`, `center`, `contains_`, `dim`, `display`, `empty`, `enclosePoints`, `generateRandom`, `Inf`, `is_bounded`, `isemptyobject`, `isequal`, `length`, `le`, `lt`, `minus`, `mtimes`, `or_op`, `origin`, `plus`, `project`, `rad`, `randPoint_`, `representsa_`, `size`, `times`, `transpose`, `uminus`, `vertcat`, `vertices_`.
- **Key Missing Tests:** Tests for all the untranslated mathematical, logical, and conversion operators are missing. For example: `test_interval_abs.m`, `test_interval_cos.m`, `test_interval_zonotope.m`, `test_interval_rdivide.m`, etc. All `testLong_*` files are also missing.

## 6. `g` (Global Functions & Classes)

A significant portion of the global helper functions, classes, and macros from `cora_matlab/global` have not been translated to `cora_python/g`.

### Missing Functions (`cora_python/g/functions`)
- **`helper` sub-directory is largely incomplete:**
  - `nn/`
  - `specification/`
  - `global/`
  - `examples/`
  - `dynamics/`
- **`verbose` sub-directory is largely incomplete:**
  - `videos/`
  - `write/`
  - `warnings/`
  - `tipoftheday/`
  - `print/`
  - `display/`
  - `ascii/`
- **`matlab` sub-directory is largely incomplete:**
  - `struct/`
  - `string/`
  - `indexing/`
  - `function_handle/`
  - `file/`
- **`codingConventions` directory is completely missing.**
